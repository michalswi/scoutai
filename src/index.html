Ollama URL Configuration

How it works:

  • Default: http://localhost:11434 - Connects to Ollama running on your local machine
  • Remote: http://192.168.1.100:11434 - Connect to Ollama on another machine in your network
  • Custom port: If Ollama runs on a different port, change 11434 to your port
  • Note: URL resets to localhost on app restart (not saved)

Quick tips:

  • Press Enter or click ğŸ”„ Refresh to update connection
  • Status will show "Ready" when connected, or error message if failed
